# 宝可梦对战游戏

一个基于命令行游玩的宝可梦对战游戏，玩家与电脑各选择3只宝可梦进行战斗。

## 游戏玩法

1. **组队阶段**：从5只宝可梦中选择3只组成你的队伍。
2. **出战选择**：从你的队伍中选择一只宝可梦出战。
3. **对战阶段**：轮流选择技能攻击对手，直到一方所有宝可梦的HP归零。

## 游戏机制

### 基础属性
每只宝可梦拥有以下属性：
- **HP**：血量，归零时宝可梦【昏厥】
- **属性**：决定克制关系和抗性
- **攻击力**：影响技能伤害
- **防御力**：减少受到的伤害
- **闪避率**：躲避攻击的概率


### 属性系统
游戏包含5种属性，具有相互克制关系：

| 属性 | 克制 | 被克制 | 属性被动 |
|------|------|--------|----------|
| 草 | 水 | 火 | 每回合回复10%最大HP |
| 火 | 草 | 水 | 每次造成伤害提升10%攻击力(最多4层) |
| 水 | 火 | 电 | 50%概率减免30%伤害 |
| 电 | 水 | 草 | 成功闪避后立即获得额外行动 |
| 土 | 电 | 水 | 免疫电属性伤害并回复20%最大HP |

**克制效果**：
- 克制方攻击被克制方：伤害×2
- 被克制方攻击克制方：伤害×0.5

### 宝可梦与技能

#### 1. 皮卡丘 (电属性)
- **技能**：
  - 十万伏特：1.4倍电属性伤害，10%概率麻痹对手
  - 电光一闪：1.0倍一般属性伤害，10%概率连击

#### 2. 妙蛙种子 (草属性)
- **技能**：
  - 种子炸弹：1.2倍草属性伤害，15%概率使对手中毒
  - 寄生种子：每回合吸取对手10%最大HP，持续3回合

#### 3. 杰尼龟 (水属性)
- **技能**：
  - 水枪：1.4倍水属性伤害
  - 护盾：下回合减伤50%

#### 4. 小火龙 (火属性)
- **技能**：
  - 火花：1.0倍火属性伤害，10%概率烧伤对手
  - 蓄能爆炎：蓄力1回合后造成3倍伤害，80%概率烧伤对手

#### 5. 地鼠 (土属性)
- **技能**：
  - 抓：1.0倍一般属性伤害
  - 挖洞：蓄力1回合后造成1.8倍地面属性伤害，蓄力期间无法被选中

### 状态效果
- **中毒**：每回合损失10%最大HP
- **烧伤**：每回合损失10点HP
- **麻痹**：50%概率无法行动
- 状态效果持续有限回合数
