## 人物速览

黄子弘凡，1999年出生，中国内地男歌手、音乐剧演员。2018年因参加湖南卫视美声竞演节目《声入人心》第一季而崭露头角，凭借其融合美声与流行的独特唱腔和阳光形象获得关注。毕业于伯克利音乐学院，是兼具学院派功底与舞台魅力的新生代艺人。

百度百科 [黄子弘凡](https://baike.baidu.com/item/%E9%BB%84%E5%AD%90%E5%BC%98%E5%87%A1 "黄子弘凡个人简介")。

## 专辑数据查询示例
``` python
# 专辑信息查询系统
album_database = {
    "如果我不是透明的": {"year": 2022, "type": "EP", "tracks": "未知" },
    "大世界的奇妙玩家": {"year": 2023, "type": "EP", "tracks": "未知" },
    "可是我最懂降落": {"year": 2024, "type": "迷你专辑", "tracks": 5 },
    "除了快乐禁止入内": {"year": 2024, "type": "全长专辑", "tracks": 10,},
    "晚一点绿灯": {"year": 2025, "type": "EP", "tracks": 5 }
}

def get_album_info(album_name):
    """查询指定专辑的基本信息"""
    if album_name in album_database:
        info = album_database[album_name]
        return (f"专辑《{album_name}》于{info['year']}年发行，"
                f"类型为{info['type']}，"
                f"收录{info['tracks']}首歌曲，")
    else:
        return "未找到该专辑信息，请确认专辑名称。"

# 示例查询
print(get_album_info("除了快乐禁止入内"))
```
